## 🌌 为何取名「众妙」？
「众妙」源自《道德经》，形容万物奥妙的根源——**“一切玄妙变化的总门”**。

作为开源组织，我们践行这一理念：
- **开放如门**：代码应如「道」，自由流通，生生不息。
- **协作为术**：众人之力，化想法为惠及所有人的工具。

本组织下的仓库包含开源库、工具及实验性项目，欢迎探索。
